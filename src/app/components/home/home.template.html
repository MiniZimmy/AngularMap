Si cette page s'affiche, c'est que vous êtes prêts à commencer a coder. Le but de ce cours-tutoriel est de vous faire découvrir Angular. Ici, vous créerez une application utilisant l'API de google afin d'afficher et d'interagir avec une carte.

Vous trouverez dans la partie droite du header un pense-bête avec toutes les propriétés les plus importantes d'Angular. Vous devriez le garder sous le coude pour les parties suivantes.

Il reste encore beaucoup de features a implémenter et vous avez maintenant toutes les cartes en mains pour le faire. Par exemple, il est pour le moment possible de créer des marqueurs mais nous n'avons aucun moyen de les gérer. Une idée d'amélioration serait d'afficher une liste de tous nos marqueurs pour pouvoir les modifier, supprimer ... C'est facile maintenant qu'on a un service dédié aux marqueurs!
Il reste aussi énormément de services fournis par Google qui permettent de jouer avec les cartes ou autre, découvrez les!

De même, il reste encore bien des choses à découvrir dans Angular et RxJS. Pour en voir une petite partie avec cette application, on peut mettre en place un [Store](https://github.com/ngrx/store) qui contiendrait nos marqueurs et places à afficher.
